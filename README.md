# Kafe Střekov ☕

Oficiální web kavárny **Kafe Střekov** – [kafestrekov.cz](https://kafestrekov.cz)

## O webu

Web kavárny Kafe Střekov obsahuje vše, co potřebuješ vědět před návštěvou:

- ☕ **Menu** – aktuální nabídka kávy, nápojů a jídla
- 📅 **Rezervace** – rezervuj si místo online
- 🕐 **Otevírací doba** – kdy nás najdeš otevřené
- 📍 **Základní informace** – adresa a kontakt
- 🖼️ **Galerie** – prohlédni si prostředí kavárny
